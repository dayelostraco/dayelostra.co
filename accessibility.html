Statement

Accessibility

This site is meant to work for everyone, and accessibility is checked automatically on every change, not audited once and forgotten.

Standard

The target conformance level is WCAG 2.1 AA, the same technical bar the U.S. federal government harmonizes Section 508 to. Color, structure, and interaction are designed against that standard.

Continuously enforced

Every change to this site runs an automated axe-core accessibility gate in continuous integration. A build with any WCAG A or AA violation does not ship. This statement therefore reflects a standard held on every commit, not a one-time pass.

What is covered

  • Contrast. Text and interface contrast verified across both the light and dark sections of the site.
  • Keyboard. Full keyboard navigation with a visible focus indicator on every interactive element.
  • Reduced motion. Animations and transitions respect the prefers-reduced-motion setting.
  • Reflow. Content reflows without horizontal scrolling down to a 320px viewport.
  • Structure. Semantic headings, landmarks, image alternatives, and a skip-to-content link.

Honest limitations

Automated testing catches a large share of accessibility issues but not all of them; some WCAG criteria require human judgment. This is an ongoing commitment maintained alongside the site, not a formal third-party audit or VPAT. If something here does not work for you, treat it as a bug worth reporting.
